1. Web Server: Sebuah web server adalah perangkat keras atau perangkat lunak yang menyimpan dan menyajikan halaman-halaman web kepada pengguna melalui internet. Ketika seseorang mengakses sebuah website, permintaan dikirimkan ke web server yang meng-host (menyimpan) website tersebut. Web server bertugas mengirimkan halaman web yang diminta ke perangkat pengguna melalui protokol HTTP atau HTTPS.

2. Domain: Domain adalah alamat unik yang digunakan untuk mengidentifikasi suatu website di internet. Contohnya, "www.nama-domain.com". Domain menyediakan cara mudah diingat dan digunakan sebagai pengganti dari alamat IP numerik yang sebenarnya dari web server. Pengguna dapat mengakses website dengan memasukkan domain ke dalam peramban web mereka.

3.Hosting: Hosting adalah layanan yang menyediakan ruang dan infrastruktur server untuk menyimpan dan membuat website dapat diakses melalui internet. Sebuah website harus di-host di server agar dapat diakses oleh pengguna. Hosting dapat dilakukan secara sendiri menggunakan server sendiri atau menggunakan layanan hosting dari penyedia hosting. Layanan hosting menyediakan server, penyimpanan data, bandwidth, dan fitur lainnya yang diperlukan agar website dapat berjalan secara online. Hubungan Antara Ketiganya:

Ketika seseorang ingin membuat website, mereka memerlukan layanan hosting untuk menyimpan konten dan data website tersebut. Setelah website di-host, pemiliknya mendapatkan alamat IP untuk server yang menyimpan website tersebut. Domain dibeli dan dihubungkan (disebut konfigurasi DNS) ke alamat IP server tempat website di-host. Ketika seseorang mengakses website dengan memasukkan domain ke dalam peramban, permintaan dikirim ke DNS (Domain Name System) untuk mengonversi domain menjadi alamat IP server yang sesuai. Setelah mendapatkan alamat IP, permintaan dikirim ke web server yang meng-host website tersebut. Web server mengirimkan halaman web yang diminta kembali ke perangkat pengguna melalui internet.
